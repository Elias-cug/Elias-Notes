### 2. 箭头函数和一般函数有啥区别
- 箭头函数是匿名函数，不能作为构造函数，不能使用new
- 箭头函数不能绑定arguments，取而代之的是用rest参数 ... 解决
    
```js
var fn = (...c) => {
    console.log(c)
}
fn(1,2,3,4); // [1,2,3,4]
```  

- 箭头函数没有原型属性
- 箭头函数不绑定 this，其 this 永远指向其上下文的 this，没有办法改变其指向，
  普通函数的 this 指向调用它的对象
